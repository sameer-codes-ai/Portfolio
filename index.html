Projects

LeetFlix [Github Link]

As part of Team CodeForSurvival, I developed LeetFlix, a Netflix & LeetCode-inspired platform featuring a curated library of WebSeries Trivia, offering users a clean, binge-worthy Ul to practice engaging trivia challenges. Engineered the full frontend using HTML, CSS, and JavaScript, replicating a Netflix-style UX with responsive cards, hover interactions, and category-based trivia browsing. Built key features including a discussion forum, leaderboard, and a powerful Admin Panel that allows adding new shows or trivia questions with a single JSON paste (auto-parsing questions, options, and answers). Developed a complete backend using Node.js and MongoDB, handling authentication, user progress, leaderboard logic, and dynamic content management.

Cryptic Bird [Github Link]

As part of Team DecentralizedDreamers, I developed a Java-based game called Cryptic Bird - an innovative spin on the classic Flappy Bird, but with added challenges like dynamic levels, moving pipes, spiked pipes, and changing backgrounds. The game integrated Blockchain technology, allowing players to enter their name and BSC blockchain address via Java Swing textboxes. Based on their score, players were rewarded with our custom DevJams Token, coded in Solidity, sent manually to their blockchain wallet after cross verification of score. We stored the player data in a .txt file and used the BEP20 token standard for crypto transactions.
